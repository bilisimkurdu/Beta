Windows 10 Cortana devre dışı bırakma

PC Windows 20 Kasım 2018 43 Like

Merhaba Muhterem ziyaretçilerimiz,
Bugün Windows 10 Cortana disable (devre dışı bırakacağımızı) nasıl edileceğini anlatacağım. Bilindiği üzere Cortana Microsoft’un asistan uygulaması olup işlevi yerine getirebilmesi için verilerinizi toparlayıp analiz edilmektedir.

Cortana’yi disable ederek gizliliğinizi bir nebzede koruyabilirsiniz.

Windows 10 Pro ve Enterpise

  1. Başlat -> gpedit.msc
  2.  Computer Configuration > Administrative Templates > Windows Components > Search yolunu takip ederek 
  3. Allow Cortana ayarını bulup çift tıklatınız disable ederek bilgisayarınız restart ediniz.

Windows 10 Home

Başlat -> Regedit -> HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Windows Searchyolunu takip ediniz.

Windows klasörünün üzerine sağtık ile tıklayarak “New->Key” yolunu takip ederek “Windows Search” değerini veriniz.

Windows Search klasörün üzerine sağtık ile “DWORD (32-bit) Value” ekliyoruz.

4. DWORD  değeri “AllowCortana” değeri “0” olmalıdır. En sonunda bilgisayarı restart ediyoruz.
